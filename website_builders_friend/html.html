Alright so HTML stands for HyperText Markup Language it is used for the basic skeleton of your web page.
It consists of tags that look like <p>(open tag) </p>(close tag). The <p> tag is used in the body to
indicate a new paraagraph. This tag will display what you type inside of the tags to you page.

<!DOCTYPE html> <html> <head> <title>TEST</title> </head> <body style="background-color:#E6E6E6"> <p>Stuff</p> </body> </html>
