What is CSS?

Before we get to know more about Bootstrap, lets take a step back and take a brief look at Cascading Style Sheets or better known as CSS.
CSS is a language that defines the style of an HTML document. CSS is very beneficial because it cuts the amount of work need to be done and
can be used to control the layout for multiple webpages. Not only does CSS help with styling your website, it also helps to define the
design and layout for different devices as well.
For more information on CSS you can visit www.w3schools.com

If CSS so good, why use Bootstrap?
Well, Bootstrap implements both HTML and CSS templates & plugins, so its the best of both worlds!
Bootstrap is easy to use and its biggest selling points: it supports responsive web design features and is mobile friendly!
In other words: with responsive web design features, the webpage can automatically adjust to look good on all devices!

Because of their similarities, there is an ongoing debate of CSS vs BootStrap.

Starting Out

What makes bootstrap able to be responsive to different devices is the <meta> tag (there is no ending tag).
What the <meta> tag does, is that it gives the browser instruction on how to control a webpage's dimensions and scaling.
Therefore, it is important to always include the <meta> tag. The <meta> tag is located inside of the <head> & </head> tags at the top of the HTML document.
Inside of the <meta> tag is where you would set your desired scaling and dimensions for the webpage.

What to include in of the <meta> tag:
    -  Charset UTF-8, represent any character in the Unicode standard and would look like this:      <meta charset="utf-8>
OR
    -  Charset UTF-16encode the entire Unicode repertoire, which looks like this:      <meta charset="utf-16>
NOTE: Only ONE can be picked

Now in a separate <meta> tag you can include the following:
    -  “width=device-width” , this sets as you guess the width of the page to the screen-width of other devices.
    -  “initial-scale=1.0” , will set the initial zoom of the webpage.
Now unlike charset, both “width=device-width” & “initial-scale=1.0” are inside of the same <meta> tag.
Which would look like this:     <meta content=“width=device-width, initial-scale=1.0”>

For more information about starting out with Bootstrap, you can visit www.w3schools.com

Bootstrap's Grid System and Containers

The Bootstrap Grid System

Bootstrap uses a grid system that is divided into twelve columns across the webpage. It is not required to used all twelve columns individually,
so you can combine columns together creating bigger columns but the sum of must equal to twelve. In total there are five different grouping avaialable:

      1: single columns, using all twelve columns individually.
      2: three big columns, so you would have three columns with a span of four ( by "span of four", I mean four individual columns grouped together).
      3: two columns, one column with a span of four and another column with a span of eight.
      4: two columns, two columns with a span of six.
      5: a single column with a span of twelve.

As you can see, all the possible groupings have a total sum of twelve columns.

The Bootstrap grid system can further be defined by four classes:

    -  xs (for phones - screens less than 768px wide)
    -  sm (for tablets - screens equal to or greater than 768px wide)
    -  md (for small laptops - screens equal to or greater than 992px wide)
    -  lg (for laptops and desktops - screens equal to or greater than 1200px wide)

Sytnax for Bootstrap Grid System

    <div class="row">      &nbs             This creates a row
       <div class="col-*-*"></div>        This creates a column, where "-*-*" is the class and the number of columns.
             .
             .                                                   Continue adding your desire columns, but remember the sum MUST BE twelve.
             .
    </div>             

Using the above sytnax, we will be creating a simple grid system with a class of sm using grouping #2:

    <div class="row">        This creates a row
       <div class="col-*-*"></div>        This creates a column with a span of 4
       <div class="col-*-*"></div>        This creates a column with a span of 4
       <div class="col-*-*"></div>        This creates a column with a span of 4
    </div>

For more information about the Bootstrap Grid System, you can visit www.w3schools.com

Containers in Bootstrap

Containers are used to pad the content inside of them, and there are two container classes that are available:
       - the .container class is a fixed width container that can have it's width edited.
       - the .container-fluid class is a container that spans the entire width of the webpage.

Syntax of .container:
    <div class="container">
       <p> your desired info </p
    </div>

Syntax of .container-fluid:
    <div class="container-fluid">
       <p> your desired info </p
    </div>

For more information about the Bootstrap Containers, you can visit www.w3schools.com
