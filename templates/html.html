The Basics of HTML

HTML stands for HyperText Markup Language. It is the basic skeleton of any web page.

HTML Documenting

Every HTML element will mostly be a combination of a start tag and a end tag, where the info is between the tags.
For exmample: <tagname> info </tagname>

Every HTML document must include the following:
    -  At the top of every HTML document is the document type declaration: <!DOCTYPE HTML>
    -  The tags <html> (starting tag) and </html> (ending tag) are the bounds of the HTML document where the body of document will be.
    -  Speaking of the body, the tags <body> and </body> is where the main information will go for your website. We will go more in-depth on this later.

For more information on HTML documenting and tags you can visit www.w3schools.com

Attributes in HTML

An HTML attribute provides more information about an HTML element. All HTML elements are capable of having attributes.
They are always located within the start tag. Example: < atrributename and then tagname> info </tagname>
Please refer to www.w3schools.com for more information on attributes.

Sytles in HTML

Sytles are similar to attributes in HTML, they can further modify HTML elements.
To set a sytle for an HTML element you would do the following: <tagname style="property:value;">
There are many sytles that can be applied, for more information on styles you can visit www.w3schools.com

Body and Paragraphs in HTML

Here we will go more in-depth on the body tags.
The tag <body> and </body> is used to deined the contents of the website which will determine what will be displayed.
This includes, but is not limited to:
    -  The text that will be displayed, or what is better known as the paragraphs in HTML.
    -  Images or Tables can also be shown as well as incorporating links to other websites and the list goes on.
Of course, we will go over the basics of some of these.

So, how do you use paragraphs in HTML?
Well, its rather simple. The start tag <p> is the beginning of the parapgraph and is closed with the end tag </p>.
Inbetween those tags would go the text that you wanted to be displayed.
For example the following:   <p>Hello World</p>   would display "Hello World" on the webpage.
For more information on HTML body and paragraphs you can visit www.w3schools.com

For practice, you can refer to the Try it Yourself section at the bottom of this page.

Headings in HTML

In HTML, by default there are six levels of headings but you are not required to stay with these defaults.
Headings consists of the tag <h#>header name</h#>, where # is a number ranging from 1 to 6.
The size of the heading are in decreasing order, with <h1> being the largest heading and <h6> being the smallest.
If the default sizes are not to your picking, then you can futher edit the size to your liking.
But we are only going over the basics, for more information on HTML headings you can visit www.w3schools.com

Links in HTML

Want to include a link to your favorite website, an intreseting article, or maybe a youtube video?
Well, worry not! Adding a link is entirely possible with HTML and here's how:
In HTML links are hyperlinks that will jump to your desired website or document when clicked on.
HTML links are recongized by the starting tag: <a> and closing tag: </a>.

Use the following syntax:
    <a href="url of desired link"> here is where you would put the link text </a>

As you can see, HTML links uses attributes, namely the "href" is a attribute.
The "href" attribute defines the address of the link.
For more information on HTML Links you can visit www.w3schools.com
Can you guess how I linked the above?

Images in HTML

Unlike most HTML elements that require both a start tag and a end tag, the HTML image tag does not require a ending tag.
Therefore, images in HTML are define with the <img> tag. But just like links in HTML, a attribute is required with each <img> tag.
To implement a HTLM image, use the follwoing syntax: <img src="url"> , where "url" is the web address of the image.

As you can see, a HTML image uses the attributes "src". The "scr" attribute defines the address of the image.
