JavaScript for when you want to add functions

JavaScript is a scripting language that can and does run in you browser.
In you HTML file you can add JavaScript in a <script> tag(see below).

The Fundamentals of JavaScript

Similar to the ongoing debate of CSS vs BootStrap, there is also a ongoing debate between JavaScript and Jquery.
Wait... I know what you must be think: "Jquery" what is that?, well don't worry we will discuss that in another tab.

JavaScript is the default scripting language for HTML and you may not know but it is always running in your browser.
No, it doesn't require any downloads and best of all its free. So, why not learn how to use it's features to your own benefits.

But what does JavaScript do exactly?

With JavaScript you will be able to edit HTML content, HTML attribute values, and you can even change HTML styles.
JavaScript is similar to coding in C. So, if you know a little bit about writing in C programming syntax, you are already one step ahead!
One example of how they are similar is that both JavaScript and C programming are capable of creating functions, objects, strings, arrays and list goes on.
An important key note to remember is that similar to coding in C, JavaScript code must end in a semicolon (;).

Where does JavaScript Go?

First, can you guess the tag that defines JavaScript? .....It should come as no suprise but it's in the name itself!
JavaScript code is defined by the starting tag <script> and the ending tag </script>.
The <script></script> tags can be place in either the <head> tags or the <body> tags. It can even be placed in both the <head> and <body> tags.
JavaScript can also be in a exterinal JavaScript file. The advangates of having JavaScript code in a external file is that separates HTML and JavaScript code.
This makes it easier to distinguish between HTML and JavaScript, therefore making maintaining the two codes less of a headache.
Finally, a big factor in having external files is that cached JavaScript files will speed up the amount of time it takes to load the page.

For more information about the basics of JavaScript, you can visit www.w3schools.com

How is JavaScript Shown?

All this juicy talk about JavaScript, but how do you show the data on your website? Easy! There are four methods of outputing information:

      1: Writing into an HTML element, using innerHTML
      2: Writing into the HTML output using document.write()
      3: Writing into an alert box, using window.alert()
      4: Writing into the browser console, using console.log()

The first option: using innerHTML. To use this method, first you would get a HTML element by using the following JavaScript method:     document.getElementById(id)
What the id attribute does is defines the HTML element. While innerHTML defines the HTML content.

Here is a simple syntax:
    <p>id="name of id"</p>
    <script>
       document.getElementById("name of id").innerHTML = "your desired data;
    </script>

The second option: using document.write(). This method should be used ONLY for testing and should NOT be on your final website.
The reason why this method is only used for testing is that using document.write() after the HTML document has been loaded, will delete the entire HTML code.
Please use with caution and only for testing purposes.

Here is a simple syntax:
    <script>
       window.alert("Insert your desired info");
    </script>

The third option: using window.alert(). Using this method will display an alert box or pop-up to display the desired information.

Here is a simple syntax:
    <script>
       window.alert("Insert your desired info");
    </script>

The fourth option: using console.log(). This method is mainly for debugging purposes and the data will not be directly displayed.
To view the data you would need to right-click on the webpage and select "Inspect" on Google Chrome or "Inspect Element" on FireFox.
From there you would select the console tab and the data will be displayed there.

Here is a simple syntax:
    <script>
       console.log("Insert your desired info");
    </script>

For more information about the Outputs of JavaScript, you can visit www.w3schools.com

Fundamentals of JavaScript Part 2

JavaScript Syntax

An important note to consider when writing JavaScript code is to remember that JavaScript is that CASE SENSITIVE!
In JavaScript, identifiers are names that are used to name variables, keywords, functions or labels. Naming identifiers follows the same rule as most programming languages.
This means that the first character in names must be a letter or a underscore (_), or dollar sign ($).

JavaScript has two definitions of values: first being "fixed values" and the second being "variable values".
In JavaScript, fixed values are named "literals" and variable values are named "variables".

For more information about the JavaScript Syntax, you can visit www.w3schools.com

Variables in JavaScript

Speaking of Variables, in JavaScript the keyword var is used to declare variables and the equal sign (=) is used to assign values to said variable.
This similar to other programming languages and is pretty straight forward.
Example:
    var x;           This declares variable x.
    x = 10;         And this will assign x to have a value of 10.

Remember that JavaScript is case sensitive, so all variables are required (like other programming languages) to have unique names.

For more information about the JavaScript Variables, you can visit www.w3schools.com

Operators and Arithmetic

Operators in JavaScript

Because JavaScript is similar to C, having operators in JavaScript is possible. Just like any other programming language, JavaScript has the following operators:
    - Addition (+)
    - Subtraction (-)
    - Multiplication (*)
    - Exponentiation (**)
    - Division (/)
    - Modulus or Remainder (%)
    - Increment (+*)
    - Decrement (--)
    - Equal (=)
    - x += y     which is equivalent to x = x + y
    - x -= y      which is equivalent to x = x - y
    - x *= y      which is equivalent to x = x * y
    - x **= y     which is equivalent to x = x ** y
    - x /= y       which is equivalent to x = x / y
    - x %= y     which is equivalent to x = x % y

The (+) operator can also be used with strings in JavaScript to concatenate two separate strings or even a number with a string.
Of course, there are many more operators but we will only be discussing those listed.

For more information about the JavaScript Operators, you can visit www.w3schools.com

Arithmetic In JavaScript

Similar to operations in JavaScript, performing arithmetics in JavaScript follows the same ideas as all other programming languages.
Numbers in the arithmetic operations are refered to as the operands. While the operations between the two operands is an operator.
Preforming arithmetics is pretty straight-forward and follows the basics of arithmetic, so just like in algebra there is a operator precedence.

For more information about the JavaScript Arithmetic, you can visit www.w3schools.com

The JavaScript Functions

A function in JavaScript is a section of code that will preform a specifc task. Similar to other programming languages, the function is only executed when called upon.

In JavaScript, functions are defined by the function keyword. Which is then followed by the function name (I'll say it again: remember JavaScript is case sensitive!),
and then followed by parentheses (). Inside of the parentheses, is where the parameters of the fucntions would go. These parameters are called the arguements of the function.

Here is the basic syntax of a fucntion:
    function functionName (parameter1, parameter2,...,parameterN) {
        .
        .
        .         "code that executes"
        .
        .
    }
