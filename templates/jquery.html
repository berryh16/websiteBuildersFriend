The Fundamentals of JQuery

What is JQuery?

JQuery is a lightweight library for JavaScript, that very much like BootStrap and CSS will make it easier for JavaScript code to be used in your website.
How JQuery accomplishes this is by wrapping multiple lines of JavaScript code that perform a task into methods in a single line of code.

There are two ways to implement JQuery.
    1. Download JQuery. There are two verisons avaialabe to download: a Production Version and a Development Verion.
        Note: Make sure to have the download files in the same directory as the pages you want to incorporate it with.
        For more information about downloading JQuery, you can visit www.w3schools.com

    2. Include it from Content Delivery Network (CDN). Two of the most popular ways of using JQuery with a CDN is with either Gooogle or Microsoft.
        To implement JQuery with Google use the following code:
               <head>
               <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js"></script>
               </head>

        To implement JQuery with Microsoft use the following code:
               <head>
               <script src="https://ajax.aspnetcdn.com/ajax/jQuery/jquery-3.4.1.min.js"></script>
               </head>

For more information starting out with JQuery, you can visit www.w3schools.com

JQuery Syntax

JQuery syntax is special that it tailored made for selecting HTML elements and performing "actions" on them.

This is done with the following basic syntax:
        $(selector).action()

Where the $ is used to define or access JQuery.
In place of the (selector) would be what element you are looking for.
The action() in JQuery is the "action" that is to be preformed on the HTML element(s).

For more information about JQuery Syntax, you can visit www.w3schools.com

For more information about the basics of JQuery, you can visit www.w3schools.com

The Magic of JQuery

Effects with JQuery

With JQuery you are able to include a varity of effects onto your website and because of that we will only be going over a few of the more popular effects.

Hide and Seek

Now you see me and now you don't.

Using JQuery, you are able to hide and show HTML elements. In JQuery to do this you would use the hide() and show() methods.
The syntax for hide() and show() methods are as follows:

   $(selector).hide(speed,callback);

   $(selector).show(speed,callback);

The speed parameter defines the speed at which the effect is displayed on the website. This parameter can be set as "slow", "fast", or it can also be in miliseconds.
While the callback parameter is in fact a function that is executed after hide() or show() methods .

What if you want to use both the hide() and show() methods?
Worry Not! In JQuery, you are able to switch between hiding and showing an element by using the toggle() method!
Just use the folllwing syntax for the toggle() method:

   $(selector).toggle(speed,callback);

For more information about hide() or show() effects with JQuery, you can visit www.w3schools.com

Sliding into those......

With this simple effect, you will easily.... Oh wait.... nevermind.

Not only is sliding HTML elements possible with JQuery, but you can also slide HTML Elements up or down!
Just like the hide() and show() methods, you are also capable of switching between sliding up or down.
In fact, the sliding methods in JQuery follow the same syntax as the hide() and show() methods.

To implement the sliding effects, the syntax is rather simple, just follow the below examples:

First: Sliding Up
   $(selector).slideUp(speed,callback);

Second: Sliding Down
   $(selector).slideDown(speed,callback);

Third: Toggling
   $(selector).slideToggle(speed,callback);

Similar to the hide() and show() methods, the parameters are the same.
The speed parameter defines the speed at which the effect is displayed on the website. This parameter can be set as "slow", "fast", or it can also be in miliseconds.
While the callback parameter is in fact a function that is executed after hide() or show() methods .

For more information about Sliding in JQuery, you can visit www.w3schools.com

I don't feel so good....

Noooooooooooo! You're fading away!!!

You guessed it! With JQuery a fading effect can also be applied onto HTML elements and just like the others fading in JQuery follows the same aspects.
With JQuery you are able to fade elements in and out of visibility separately or by switching between the two.

To implement the fading effects, the syntax is just like the rest:

First: Sliding Up
   $(selector).fadeIn(speed,callback);

Second: Sliding Down
   $(selector).fadeOut(speed,callback);

Third: Toggling
   $(selector).fadeToggle(speed,callback);

It should come as no surpise now; the parameters are the same.
The speed parameter defines the speed at which the effect is displayed on the website. This parameter can be set as "slow", "fast", or it can also be in miliseconds.
While the callback parameter is in fact a function that is executed after hide() or show() methods .

BUT! There is one new method! In JQuery it is possible to fade a HTLM element to a certain opactiy.
By To implement the fadeTo() method, the syntax is just like the rest but with one parameter added:

   $(selector).fadeTo(speed,opacity,callback);

Where the opacity parameter specifies the fading to a given opacity between the ranges of 0 and 1.

For more information about Fading in JQuery, you can visit www.w3schools.com
